-instrucciones de ejecución:

java Juego

-modo interfaz grafica:
java Juego -gui

como jugar:
1. seleccionar la opcion "iniciar juego"
2. ingresar el nombre del equipo
3. ingresar los nombres de los 3 jugadores
en cada turno
3. seleccionar jugador (1-3)
4. seleccionar tipo de carta (aire, tierra o agua)
5. repetir para formar un combo de 3 cartas
6. ingresar la cantidad de ataques del combo
7. repetir hasta derrotar al oponente o quedarse sin energia
